# Loan Approval Prediction

A machine learning project that predicts whether a loan application will be approved based on applicant details such as income, credit score, employment status, and other financial factors. This project demonstrates data preprocessing, feature scaling, and model building for accurate predictions.

## Features
- Predicts loan approval status (Approved / Not Approved)
- Handles both numerical and categorical input features
- Uses standard scaling for better model performance
- Simple and interactive interface for predictions

## Tech Stack
- **Programming Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Modeling:** Classofication
- **Interface:**  Gradio for interactive predictions
